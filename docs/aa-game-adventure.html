AA Game Adventure

This static doc captures the current state of the playable Vite prototype hosted inside JoshHub. It explains how to launch, where to find the map data, and how to refresh the build.

Play quickstart

  • Use the Apps dashboard Play button to open /games/aa-game-adventure/index.html.
  • Arrow keys/WASD move the player; touch controls appear on mobile automatically.
  • Interact with NPCs, loot chests, and portal to room2 while the game saves locally.

Assets & map data

The tutorial map that ships with this build lives at /games/aa-game-adventure/src/data/maps/tutorial.v1.json, which describes the tiles, NPC guard, treasure chest, and portal metadata.

The remainder of the assets (sprites, audio, CSS) live next to the compiled bundle inside the same public folder.

Development workflow

  • Source code lives in Family/AModuleDnd/AModuleDnd outside of JoshHub.
  • Run npm run build in that Vite project and copy the resulting dist/ contents into JoshHub/public/games/aa-game-adventure to refresh the playable bundle.
  • Document any map, NPC, or save-slot changes here so the dashboard notes always line up with the live bundle.
