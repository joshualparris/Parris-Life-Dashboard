Panoramas

This folder lists available panorama collections. Next.js does not provide directory listings by default, so this index page is used to surface available panos.

Looking for specific panos? Use the links below or drop exported equirectangular JPGs into /public/panos/<collection>/.

  • Boundary Road (viewer placeholder)

How to add panos

  1. Create a folder under public/panos/, e.g. public/panos/boundary-road/.
  2. Copy your equirectangular images (JPG/PNG) into that folder.
  3. Add an index.json if you want hotspots/ordering, or add an index.html viewer.
  4. I can integrate a small viewer (Pannellum) to auto-detect images — tell me if you want that.
